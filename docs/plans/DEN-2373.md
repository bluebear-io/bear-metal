# DEN-2373 — Change bear metal startup log message to "hello! i'm up!"

## Goal

Replace the bear-metal manager's themed startup log message with the literal
text `hello! i'm up!`, as requested in DEN-2373.

## Scope

- Update the startup log line emitted after the health server begins listening
  in `src/manager/index.ts` from the current bear-themed message to
  `hello! i'm up!`.

## Out of scope

- The "health server listening" structured log line on the preceding line is
  left untouched (it carries the port and is used by ops).
- The shutdown log message is not affected — the ticket only references the
  startup message.

## Files touched

- `src/manager/index.ts` — change the startup `logger.info` message string.
- `docs/plans/DEN-2373.md` — this plan.

## Verification

- `grep` confirms the old message string is gone and the new string is the
  only startup log message.
- `npm run build` / `npm test` (if applicable) pass locally in CI.
