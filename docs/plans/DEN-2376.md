## Goal

Change the bear-metal manager startup log message from `hello! i'm up!` to `🐻 Bear Metal is awake and hungry for tickets — let's ship some code!`.

## Files touched

## Verification

- `rg "hello! i'm up!" bear-metal/src` returns no matches.
- `rg "Bear Metal is awake" bear-metal/src` returns the updated `logger.info` line.
- Log-only change with no behavioral impact; no tests required.
