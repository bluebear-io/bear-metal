# DEN-2444 — Add OpenSSF Scorecard badge to bear-metal

## Goal

Add the OpenSSF Scorecard workflow and a live Scorecard badge to `bluebear-io/bear-metal`,
matching the setup used in `bluebear-io/baloo-bear`.

## Files touched

- `.github/workflows/scorecard.yml` — standard `ossf/scorecard-action` workflow.
  - SHA-pinned actions.
  - Triggers: weekly cron (`23 4 * * 1`), `push` to `main`, `branch_protection_rule`.
  - `publish_results: true` so the badge endpoint at `api.securityscorecards.dev` is populated.
  - SARIF uploaded as an artifact and to the GitHub code-scanning dashboard.
  - `actions/upload-artifact` pinned to v4 (v3 is deprecated by GitHub).
- `README.md` — adds the live OpenSSF Scorecard badge next to the existing CodeQL badge.

## Verification

- Workflow appears under Actions tab after merge to `main`.
- Badge resolves at `https://api.securityscorecards.dev/projects/github.com/bluebear-io/bear-metal/badge`.
- SARIF results appear under Security → Code scanning.
